== NAME

coresched - manage core scheduling cookies for tasks

== SYNOPSIS

*coresched* [options] *-p* _pid_

*coresched* [options] \-- _command_ [_argument_...]

== DESCRIPTION
The *coresched* command is used to retrieve or modify the core scheduling cookies of a running process given
its _pid_, or to spawn a new _command_ with core scheduling cookies.
